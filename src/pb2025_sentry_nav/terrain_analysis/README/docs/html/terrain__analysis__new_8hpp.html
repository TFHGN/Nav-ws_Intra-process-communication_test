详细描述

地形分析节点头文件 - 定时器驱动的可组合节点版本

作者
Hongbiao Zhu
日期
2024

该文件定义了用于机器人导航的地形分析节点类。 主要功能包括：

  • 点云数据采集与预处理
  • 基于体素的地形网格管理
  • 地面高程估计
  • 障碍物检测与动态障碍物过滤
  • 地形可通行性评估

架构特性：

  • 支持进程内通信（Intra-Process Communication）
  • 100Hz定时器驱动的非阻塞处理
  • 适配 Nav2 组件容器部署
  • 支持实时地形滚动更新
